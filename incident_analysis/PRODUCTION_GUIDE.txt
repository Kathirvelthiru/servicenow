HOW TO USE
==========

INSTALLATION:
pip install -r requirements_prod.txt

INTERACTIVE MODE (Single Incident):
python incident_pattern_matcher_prod.py interactive

BATCH MODE (Multiple Incidents):
python incident_pattern_matcher_prod.py batch -i incidents.json -o results.json

SAMPLE BATCH INPUT FILE (incidents.json):
[
  {
    "title": "Database connection timeout",
    "description": "Users unable to log in during peak hours",
    "category": "Database",
    "sub_category": "Performance"
  },
  {
    "title": "Network connectivity issue",
    "description": "Office internet drops frequently",
    "category": "Network",
    "sub_category": "Connectivity"
  }
]

OUTPUT:
- Logs saved to: incident_matcher.log
- Matches saved to: incident_matches.db (SQLite)
- Batch results saved to: matches_YYYYMMDD_HHMMSS.json
